# Cybersecurity — Integrated with AI

## Summary

This is an 18-week online certificate programme from the NUS School of Computing covering the fundamentals of cybersecurity, with dedicated modules on how AI and generative AI reshape both attack and defence. It treats cybersecurity not as an IT specialty but as a broader security discipline — one where threats can originate anywhere in the world and where the defender's toolkit increasingly includes machine learning alongside traditional cryptography, network controls, and access control.

The programme is aimed at working professionals — particularly those in IT services, banking and financial services, network security, and management consulting — who want to incorporate cybersecurity fundamentals into their function or organisation. The stated motivation is the rapid digitisation of the global economy, a corresponding growth in digital crime, and a persistent global talent gap in cybersecurity. It assumes learners want both conceptual grounding and practical exposure to real tools and attack scenarios.

Learners come away able to explain the fundamentals of cybersecurity and the C-I-A (confidentiality, integrity, availability) security requirements; describe core techniques including encryption, authentication, and secure-channel construction; recognise well-known attacks on hosts, networks, and web servers and the counter-measures against them; and apply best practices to safeguard systems. The AI-specific objectives (inferred) include understanding how machine learning and deep learning are used for threat detection, vulnerability assessment, and security automation, and understanding the new risks introduced by generative AI.

The pedagogical approach blends pre-recorded NUS faculty lectures with live online sessions, faculty demo videos on industry tools (Wireshark, Nmap, Metasploit, Nessus), case studies drawn from real incidents (Mozilla Firefox, Dunkin' Donuts, Google Chrome, GitLab, Singapore's national cybersecurity plan), assignments, discussion boards, and a capstone project. Three modules — Cloud Security, AI in Cybersecurity, and Generative AI and Cybersecurity — are delivered live online rather than as recordings.

The curriculum moves from foundations (why cybersecurity is hard, classical and modern cryptography, authentication) through applied security domains (protocols, network, web, system, malware) and into contemporary concerns (cloud, AI, generative AI, ethical hacking, governance). The capstone applies these learnings to a real-world scenario.

## Topics

Introduction to Cybersecurity — Establishes why cybersecurity matters as a distinct discipline and why achieving security is fundamentally difficult, setting the framing for the rest of the programme.

Cryptography — Covers encryption, decryption, and keys; classical ciphers; the formal definitions and properties of cryptosystems; and the principle that security should rest on key secrecy rather than algorithm secrecy (Kerckhoffs' principle versus security by obscurity).

Modern Ciphers — Introduces the two main families of symmetric encryption used today: stream ciphers and block ciphers.

Entity Authentication (Password) — Treats the problem of proving who a party is, with focus on passwords as the dominant mechanism and multi-factor authentication as the standard hardening approach.

Public Key System & Cryptographic Hash — Introduces public-key cryptography (PKC) and cryptographic hash functions, including keyed hashes, as the building blocks for the authentication and integrity mechanisms that follow.

Data Origin Authentication & Public Key Infrastructure — Covers how to authenticate the source of a message using MACs and digital signatures, and the supporting machinery of public key distribution (PKD) and public key infrastructure (PKI).

Protocol Security — Strong authentication protocols, key exchange protocols, authenticated key exchange, and how these combine to secure a communication channel end-to-end.

Network Security — The network layer; attacks on name resolution; denial-of-service attacks; practical network security tools (Wireshark and Nmap); and firewalls as a primary perimeter control.

Web Security — Web threat models and the major classes of web attack: UI-based attacks, cross-site scripting (XSS), cross-site request forgery (CSRF), and SQL injection.

System Security and Access Control — Access control as a concept, how it is represented, how it works concretely in UNIX/Linux, and the mechanisms of controlled invocation and privilege elevation.

Attacks, Malware & Mitigation — The attack framework and cyber kill chain; reconnaissance; vulnerability scanning; automated exploitation; and malware along with its defences.

Cloud Security (live online) — Cloud concepts, architecture and design; cloud data security; cloud platform and infrastructure security; cloud application security; and best practices and technology for securing cloud environments.

AI in Cybersecurity (live online) — AI algorithms and techniques applied to security; machine learning for threat detection; deep learning for cybersecurity; AI for vulnerability assessment and penetration testing; and AI for incident response and security automation.

Generative AI and Cybersecurity (live online) — Introduction to generative AI; how it increases risks and enables novel attacks; how it can also be used for evolved threat identification; and the future of generative AI and automation in security.

Ethical Hacking — Introduction to ethical hacking, network scanning, system hacking, and session hijacking — the offensive techniques a defender needs to understand.

Ethics, Governance and Future of Cybersecurity — Security principles and governance, codes of ethics, best practices across network/cloud/IoT security, and emerging threats and technologies.

Capstone Project — A culminating project applying the programme's learnings to a real-world scenario (inferred: scope and topic are learner-defined within the cybersecurity domain).
